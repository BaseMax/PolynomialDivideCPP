# Polynomial Algorithm

Implement algorithm of **divide** operator of **polynomial** in **C++**.

## Using

**C++:**
```bash
$ g++ polynomial.cpp -o out.cpp
$ ./out.cpp
```

**C:**
```bash
$ gcc polynomial.c -o out.c
$ ./out.c
```
